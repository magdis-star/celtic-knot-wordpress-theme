## Where to Edit Homepage Content

### Step 1: Go to Homepage Settings
1. Log in to WordPress Dashboard
2. Look in the left sidebar for **"Homepage Settings"** (it has a home icon)
3. Click on it

### Step 2: Edit the Content
You'll see organized sections with easy-to-edit fields:

---

## Currently Editable Sections

### 1. Hero Section (Top of Homepage)
**Fields you can edit:**
- **Hero Title** - Main headline at top of page
  - Default: "Your Love Story. Woven with Meaning and Celtic Spirit."

- **Hero Description** - Subheading text below title
  - Default: "Licensed Humanist Officiant Kerstin Sandstrom specializes in bespoke ceremonies..."

- **Hero Button Text** - Text on the call-to-action button
  - Default: "Inquire & Check Availability"

---

### 2. My Story Section
**Fields you can edit:**
- **My Story - Photo** - Click "Add Image" to upload Kerstin's photo
  - Falls back to default photo if not set

- **My Story - Label** - Small text above the title
  - Default: "Meet Kerstin"

- **My Story - Title** - Main heading for this section
  - Default: "The Officiant Behind the Knot"

- **My Story - Bio Content** - Full biography text with rich text editor
  - Can add paragraphs, bold text, links, etc.
  - Falls back to default bio if not set

- **My Story - Button Text** - Text for the button
  - Default: "Start the Conversation Today"

---

## How to Save Changes

1. Make your edits in the fields
2. Scroll to the top or bottom of the page
3. Click the **"Publish"** or **"Update"** button
4. Visit your homepage to see the changes!

## Tips for Editing

### Text Fields
- Keep titles concise and punchy
- Avoid making descriptions too long

### Image Fields
- Click "Add Image" button
- Upload new image or select from Media Library
- Recommended image sizes:
  - My Story photo: 400x400px or larger
  - Should be good quality but not huge file size

### WYSIWYG Editor (Bio Content)
- Use the visual editor just like Microsoft Word
- **Bold** important phrases
- Break content into paragraphs
- Don't worry about HTML - the editor handles it

---

## Troubleshooting

**Q: I don't see "Homepage Settings" in the sidebar**
- Make sure ACF plugin is activated (Plugins > Installed Plugins)
- Try refreshing the page

**Q: Changes aren't showing on the homepage**
- Make sure you clicked "Publish" or "Update"
- Try clearing browser cache (Ctrl+F5 or Cmd+Shift+R)
- Some hosts cache pages - wait a minute and refresh

**Q: I want to go back to the default text**
- Just delete your custom text from the field
- Leave it empty
- Save, and it will show the original default text
