# SEO Features Implemented

Your WordPress theme now includes comprehensive SEO optimization to help your client rank well in search engines.

## Built-in SEO Features

### 1. Meta Tags (Automatic)
Every page automatically includes:
- **Meta Description** - Generated from page/post content
- **Robots Tag** - Tells search engines to index and follow links
- **Canonical URL** - Prevents duplicate content issues

### 2. Open Graph Tags (Facebook Sharing)
When someone shares on Facebook, they'll see:
- Proper title and description
- Featured image (or logo as fallback)
- Website name
- Locale set to English Canada

### 3. Twitter Card Tags
Optimized previews when shared on Twitter/X:
- Large image card format
- Title and description
- Featured image

### 4. JSON-LD Schema Markup

#### LocalBusiness Schema (Homepage)
Search engines understand this is a local wedding celebrant business in Ontario with:
- Business name, description, contact info
- Service area: Ontario
- Price range: $300-$895
- Two main service offerings:
  - Deluxe Wedding Ceremony ($895 CAD)
  - Full Service Wedding ($650 CAD)
- Instagram profile link
- Geographic location (Ontario, Canada)

#### Article Schema (Blog Posts)
Each blog post includes structured data with:
- Article title and description
- Publication and modification dates
- Author information
- Publisher details with logo

### 5. Geographic Targeting
Every page includes:
- Region: CA-ON (Ontario, Canada)
- Placename: Ontario
This helps with local search rankings.

### 6. XML Sitemap
WordPress automatically generates a sitemap at:
`https://yoursite.com/wp-sitemap.xml`

The sitemap includes:
- All pages
- All blog posts
- All testimonials
- All services

### 7. URL Optimization
- Trailing slashes added consistently (SEO best practice)
- Clean permalink structure ready

### 8. Image Optimization
Custom image sizes defined:
- Hero images: 1920x1080
- Gallery images: 800x600
- Blog thumbnails: 600x400
- Social sharing images: 1200x630

## What Your Client Needs to Do

### For Every New Blog Post:
1. **Add a featured image** - This will show in social media previews
2. **Write a custom excerpt** - Or it will auto-generate from first 30 words
3. **Add alt text to images** - In the image upload dialog, fill the "Alt Text" field
4. **Use headings properly** - H2 for main sections, H3 for subsections

### Optional SEO Enhancements:

#### Recommended Plugin: Yoast SEO (Free)
After installing this plugin, your client can:
- Customize meta descriptions for each page/post
- Preview how pages look in Google search results
- Get SEO suggestions (readability, keyword optimization)
- Generate XML sitemaps with more control

**To Install:**
1. Go to WordPress Dashboard > Plugins > Add New
2. Search for "Yoast SEO"
3. Click Install, then Activate

#### Alternative: Rank Math (Free)
Similar features to Yoast, some prefer the interface.

## Google Search Console Setup

**Highly Recommended** - This lets you monitor how the site performs in Google search:

1. Go to https://search.google.com/search-console
2. Sign in with Google account
3. Add property: Enter the website URL
4. Verify ownership (multiple methods available)
5. Submit sitemap: `https://tiethecelticknot.ca/wp-sitemap.xml`

### Benefits:
- See which keywords bring visitors
- Monitor search rankings
- Get alerts about technical issues
- Request re-indexing after updates

## Performance Tips

### Image Optimization Plugin
Install **Smush** (free) or **ShortPixel**:
- Automatically compresses uploaded images
- Reduces page load times
- Better SEO rankings (Google favors fast sites)

### Caching Plugin
Install **WP Super Cache** or **W3 Total Cache**:
- Makes pages load faster
- Reduces server load
- Improves Google rankings
