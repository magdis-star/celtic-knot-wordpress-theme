# Quick Setup Checklist for Client

## Before You Start
- [ ] Have WordPress installed and ready
- [ ] Have admin access to WordPress dashboard

## Installation Steps (30-45 minutes)

### Step 1: Upload Theme (5 min)
- [ ] Upload `celtic-knot-wordpress-theme.zip` to WordPress
- [ ] Go to Appearance > Themes > Add New > Upload Theme
- [ ] Activate the theme

### Step 2: Install Plugins (10 min)
- [ ] Install "Advanced Custom Fields" plugin
- [ ] Activate ACF plugin
- [ ] (Optional) Install Yoast SEO

### Step 3: Set Up Custom Fields (10 min)
- [ ] Go to Custom Fields > Add New
- [ ] Create "Homepage Settings" field group
- [ ] Add all hero section fields (see README.md for full list)
- [ ] Add services, gallery, testimonials fields
- [ ] Add contact info fields (email, phone)
- [ ] Set location to "Options Page = Homepage Settings"
- [ ] Publish field group

### Step 4: Create Homepage (5 min)
- [ ] Go to Pages > Add New
- [ ] Title: "Home"
- [ ] Publish (leave content empty)
- [ ] Go to Settings > Reading
- [ ] Set "A static page" as homepage
- [ ] Select "Home" page
- [ ] Create "Blog" page for blog posts
- [ ] Save changes

### Step 5: Set Up Menu (5 min)
- [ ] Go to Appearance > Menus
- [ ] Create "Primary Menu"
- [ ] Add custom links:
  - Services (#services)
  - Gallery (#gallery)
  - Testimonials (#testimonials)
  - Blog (link to blog page)
  - Contact (#contact)
- [ ] Assign to "Primary Menu" location
- [ ] Save

### Step 6: Upload Logo (2 min)
- [ ] Go to Appearance > Customize > Site Identity
- [ ] Upload logo (logo-heart.png)
- [ ] Set site title: "Tie the Celtic Knot"
- [ ] Set tagline: "Humanist Wedding Celebrant"
- [ ] Publish

### Step 7: Fill Homepage Content (10 min)
- [ ] Go to Homepage Settings in sidebar
- [ ] Fill in hero title and description
- [ ] Upload hero background image
- [ ] Fill in all section labels and titles
- [ ] Add contact email and phone
- [ ] Update/Save

### Step 8: Add Your Content
- [ ] Add 3-5 Services (Services > Add New)
- [ ] Add 3-5 Testimonials (Testimonials > Add New)
- [ ] Add 3-6 Gallery Images (Gallery Images > Add New)
- [ ] Write your first blog post (Posts > Add New)

## You're Done!

Visit your site and see your beautiful new wedding celebrant website!

## Next Steps
- Write blog posts regularly
- Update testimonials as you get new ones
- Add new gallery photos after ceremonies
- Update services as needed
